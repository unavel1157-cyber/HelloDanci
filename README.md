# HelloDanci

🇨🇳 中文介绍 | 🇺🇸 English Introduction


## 中文介绍

HelloDanci 是一款基于 SwiftUI 开发的单词学习 App。特点是“聚焦，交互，简单”

项目受到 Apple Liquid Glass 设计语言启发，尝试将现代 iOS 视觉设计理念融入学习工具中。

通过卡片式交互、手势操作以及流畅动画，让用户获得更加自然和沉浸式的单词学习体验。

为用户体验做深度优化，我是长期背单词软件的重度使用者，根据背单词用户需求，以人为本，去掉繁琐操作，冗余功能。


## English Introduction

HelloDanci is a vocabulary learning application built with SwiftUI. Base on "focus, UX, Simple"

Inspired by Apple's Liquid Glass design language, this project explores modern iOS UI design principles and interactive experiences.

The app focuses on card-based learning, gesture interactions, and smooth animations to provide a simple and immersive vocabulary learning experience.

Carry out in-depth optimization for user experience. As a heavy long-term user of vocabulary memorization apps, I design with users at the core based on the demands of vocabulary learners, eliminating cumbersome operations and redundant features.
